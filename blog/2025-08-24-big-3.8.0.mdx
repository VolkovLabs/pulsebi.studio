# Business Intelligence 3.8.0: ?

In a world driven by data, staying ahead requires tools that not only keep pace but redefine the boundaries of what's possible. We are thrilled to introduce **Business Intelligence 3.8.0**, a transformative release packed with innovative features, intuitive UI enhancements, and powerful performance optimizations for the [Business Intelligence platform](/docs/).

This update is designed to empower users with smarter, faster, and more secure data-driven decision-making. Let’s explore the key updates and improvements that make this release a game-changer for your workflows.

## What’s New in Business Intelligence 3.8.0?

Business Intelligence 3.8.0 brings significant advancements to the platform, including enhanced alert and report management, cutting-edge performance optimizations, and user-friendly UI updates. With new scheduling flexibility, streamlined engine maintenance capabilities, and updated security features, this release ensures a more efficient, robust, and seamless experience for all users.

Dive into the details of each innovation below.

- **Engine Management:** Added start and pause functionalities for engines, enabling effortless maintenance and better control during operational downtimes.
- **Alert Management:** Relocated dashboard variable repeat options to the target section alongside dashboard and panel settings, simplifying configuration and _enhancing usability_.
- **Alert Management:** Moved Event configuration to a dedicated Event Configuration tab, reducing complexity and enabling _more intuitive alert setup_.
- **Report Details:** Enhanced report functionality by adding a download report image overlay, similar to alert screenshots, for _seamless sharing and documentation_.
- **Reports:** Introduced additional pages to the reports list, providing clearer feedback and a _more polished user interface_ when no data is available.
- **Timezone Scheduling:** Added the ability to select timezones for alert and report schedules, ensuring _accurate timing_ across global teams and operations.
- **Performance:** Optimized JSON viewer and UI components for _faster load times_ and smoother interactions, improving overall user experience in data-heavy workflows.
- **Auto Thresholds:** Updated auto-thresholding capabilities with refined algorithms for more accurate anomaly detection and adaptive alerting.
- **Actions:** Introduced start and pause options to temporarily suppress notifications, offering greater flexibility during maintenance or testing phases.

Let’s dive deeper into how these features can revolutionize your Grafana workflows!

### Engine Management: Effortless Control

Managing your data engines has never been easier. With the addition of **start and pause functionalities**, you can now seamlessly handle maintenance tasks and gain better control during operational downtimes. This feature minimizes disruptions and ensures your systems are always ready when you need them.

### Alert Management: Simplified and Intuitive

We’ve reimagined alert management to make it more user-friendly and efficient:

- **Relocated Dashboard Variable Repeat Options:** These settings have been moved to the target section alongside dashboard and panel configurations, simplifying setup and **enhancing usability**.
- **Dedicated Event Configuration Tab:** Event settings now have their own tab, reducing complexity and enabling a **more intuitive alert setup process**. This streamlined approach helps you configure alerts with precision and ease.

### Report Details: Seamless Sharing

Reporting just got better with the addition of a **download report image overlay**, mirroring the functionality of alert screenshots. This enhancement allows for **effortless sharing and documentation**, making it easier to communicate insights with your team or stakeholders.

### Reports: Polished User Interface

We’ve introduced **additional pages to the reports list**, providing clearer feedback and a **more polished user interface** when no data is available. This update ensures a smoother and more professional experience, even in edge cases.

### Timezone Scheduling: Global Precision

For teams operating across multiple regions, we’ve added the ability to **select timezones for alert and report schedules**. This ensures **accurate timing** and synchronization, no matter where your team or operations are based, fostering seamless collaboration across global workflows.

### Performance: Faster and Smoother

Performance is at the heart of this release. We’ve optimized the **JSON viewer and UI components** for **faster load times** and smoother interactions. These improvements enhance the overall user experience, especially in data-heavy workflows, ensuring you spend less time waiting and more time analyzing.

### Auto Thresholds: Smarter Anomaly Detection

Our updated **auto-thresholding capabilities** feature refined algorithms for **more accurate anomaly detection** and adaptive alerting. This means you’ll receive more relevant notifications, helping you focus on critical insights without being overwhelmed by false positives.

### Actions: Flexible Notification Control

We’ve introduced **start and pause options** for actions, allowing you to temporarily suppress notifications during maintenance or testing phases. This added flexibility ensures you maintain control over your workflows without unnecessary interruptions.

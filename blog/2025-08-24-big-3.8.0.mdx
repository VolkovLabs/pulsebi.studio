# Business Intelligence 3.8.0: Redefining Data-Driven Excellence

In a world powered by data, staying ahead demands tools that push the boundaries of innovation. We’re excited to unveil **Business Intelligence 3.8.0**, a groundbreaking release for the [Business Intelligence platform](/docs/) packed with cutting-edge features, intuitive UI improvements, and robust performance enhancements.

This update empowers users with smarter, faster, and more secure ways to make data-driven decisions. Join us as we explore the key innovations in this release and how they can transform your workflows.

## What’s New in Business Intelligence 3.8.0?

Business Intelligence 3.8.0 introduces powerful advancements, from enhanced alert and report management to performance optimizations and user-friendly design updates. With flexible scheduling, streamlined engine controls, and strengthened security, this release delivers a seamless and efficient experience for all users.

Here’s a snapshot of the key updates:

- **Engine Management:** New start and pause functionalities for effortless maintenance and control during operational downtimes.
- **Alert Management:** Simplified configuration with relocated dashboard variable repeat options and a dedicated Event Configuration tab for intuitive setups.
- **Report Details:** Added a download report image overlay, mirroring alert screenshots, for seamless sharing and documentation.
- **Reports UI:** Enhanced reports list with additional pages and clearer feedback for a polished interface, even when no data is available.
- **Timezone Scheduling:** Select timezones for alert and report schedules to ensure accurate timing across global teams.
- **Performance:** Optimized JSON viewer and UI components for faster load times and smoother interactions in data-intensive workflows.
- **Auto Thresholds:** Refined algorithms for more precise anomaly detection and adaptive alerting.
- **Actions:** Start and pause options to temporarily suppress notifications during maintenance or testing.

Let’s dive into how these features can elevate your Grafana workflows.

### Engine Management: Seamless Control

Managing data engines is now simpler than ever. With new **start and pause functionalities**, you can handle maintenance tasks and control operations during downtimes with minimal disruption. Keep your systems primed and ready whenever you need them.

### Alert Management: Streamlined and Intuitive

We’ve redesigned alert management for greater efficiency and ease of use:

- **Relocated Dashboard Variable Repeat Options:** Moved to the target section alongside dashboard and panel settings, simplifying configuration and **boosting usability**.
- **Dedicated Event Configuration Tab:** Event settings now reside in a separate tab, reducing clutter and enabling a **more intuitive setup process** for precise alert customization.

### Report Details: Effortless Sharing

Reporting is now more versatile with a **download report image overlay**, akin to alert screenshots. This feature facilitates **easy sharing and documentation**, helping you communicate insights effectively with teams and stakeholders.

### Reports UI: Polished and Professional

We’ve enhanced the **reports list** with additional pages and clearer feedback, ensuring a **refined user interface** even when no data is available. This update delivers a smoother, more professional experience in all scenarios.

### Timezone Scheduling: Precision for Global Teams

For distributed teams, we’ve added the ability to **select timezones for alert and report schedules**. This ensures **accurate timing and synchronization**, enabling seamless collaboration across regions and time zones.

### Performance: Faster, Smoother Interactions

Performance is a cornerstone of this release. Optimizations to the **JSON viewer and UI components** deliver **faster load times** and fluid interactions, enhancing the user experience in data-heavy environments so you can focus on insights, not delays.

### Auto Thresholds: Smarter Anomaly Detection

Our updated **auto-thresholding capabilities** feature advanced algorithms for **more accurate anomaly detection** and adaptive alerting. Stay focused on critical insights with fewer false positives and more relevant notifications.

### Actions: Flexible Notification Management

New **start and pause options** for actions let you temporarily suppress notifications during maintenance or testing. This added flexibility ensures uninterrupted control over your workflows without unnecessary alerts.

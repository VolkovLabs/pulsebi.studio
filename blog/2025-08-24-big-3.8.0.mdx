# Business Intelligence 3.8.0: ?

In a world driven by data, staying ahead requires tools that not only keep up but push boundaries. We are thrilled to unveil **Business Intelligence 3.8.0**, a release packed with groundbreaking features, intuitive UI enhancements, and performance optimizations to the [Business Intelligence platform](/docs/).

Designed to empower users with smarter, faster, and more secure data-driven decision-making. Let’s dive into the key updates and improvements in this release.

## What’s New in Business Intelligence 3.8.0?

This release brings significant advancements to Business Intelligence platform with enhanced alert and report management, improved performance optimizations, and user-friendly UI updates. It also introduces new scheduling flexibility, streamlined engine maintenance capabilities, and updated security features, ensuring a more efficient and robust experience for users.

Dive into the details of each innovation below.

- **Engine Management:** Added start and pause functionalities for engines, enabling effortless maintenance and better control during operational downtimes.
- **Alert Management:** Relocated dashboard variable repeat options to the target section alongside dashboard and panel settings, simplifying configuration and _enhancing usability_.
- **Alert Management:** Moved Event configuration to a dedicated Event Configuration tab, reducing complexity and enabling _more intuitive alert setup_.
- **Report Details:** Enhanced report functionality by adding a download report image overlay, similar to alert screenshots, for _seamless sharing and documentation_.
- **Reports:** Introduced additional pages to the reports list, providing clearer feedback and a _more polished user interface_ when no data is available.
- **Timezone Scheduling:** Added the ability to select timezones for alert and report schedules, ensuring _accurate timing_ across global teams and operations.
- **Performance:** Optimized JSON viewer and UI components for _faster load times_ and smoother interactions, improving overall user experience in data-heavy workflows.
- **Auto Thresholds:** Updated auto-thresholding capabilities with refined algorithms for more accurate anomaly detection and adaptive alerting.
- **Actions:** Introduced start and pause options to temporarily suppress notifications, offering greater flexibility during maintenance or testing phases.

Let’s dive deeper into how these features can revolutionize your Grafana workflows!

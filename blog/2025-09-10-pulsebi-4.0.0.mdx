# PulseBI Platform 4.0.0: Elevating Data-Driven Decision Making

In today’s data-driven world, staying ahead requires tools that deliver innovation and efficiency. We’re thrilled to introduce **PulseBI Platform 4.0.0**, a transformative update to the [platform](/docs/) packed with advanced features, intuitive UI enhancements, and significant performance improvements.

This major release introduces groundbreaking updates to PulseBI Studio and Engine, focusing on enhanced user experience, advanced security measures, and significant performance improvements. Key features include revamped alert and reporting capabilities, streamlined UI workflows, and modernized engine architecture, ensuring greater efficiency and scalability for all users.

## What’s New in PulseBI Platform 4.0.0?

Here’s a quick overview of the highlights:

- **Alerts:** Added functionality to copy alerts to another engine, simplifying cross-environment management of alert configurations.
- **Alert Evaluations:** Enhanced visibility by displaying selected field thresholds and threshold units, offering greater precision in alert customization.
- **Report Details:** Now display failed reports in history, providing better visibility into reporting issues for quick resolution.
- **Actions Enhancements:** Display alerts and reports linked to actions, enabling contextual tracking and improved decision-making.
- **AI Interactivity:** Introduced MCP server integration for enhanced AI-driven insights and smarter data processing.
- **Auto Update:** Moved Auto-update logic to the Home screen for easier access and a more intuitive user experience.
- **Organizations:** Display pending users in Organization settings, ensuring streamlined user management for administrators.
- **Renderer Microservice:** Added renderer JWT authentication for improved security and safer data rendering workflows.
- **Compatibility with Previous Versions:** Removed Feature Flags to ensure a cleaner architecture.
- **Refactoring:** Refactored modules and optimized performance for faster processing and a more responsive engine.
- **Dashboards:** Improved annotations scopes and permissions, providing granular control over dashboard interactions.
- **Security:** Enhanced interaction with security fields to ensure robust protection of sensitive data and user information.

Let’s dive deeper into these features and see how they can elevate your experience with Grafana and beyond.
